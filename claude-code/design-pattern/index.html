대화 히스토리와 학습된 지식이 다층 메모리 스택에 축적되는 과정

INPUT
대화 히스토리 + 이전 세션 + CLAUDE.md 파일
PROCESS

피드백 | 패턴 | 워크플로우 | 기술 발견
OUTPUT
풍부한 컨텍스트 system-reminder로 주입
핵심 인사이트: 세션 종료 시 백그라운드 서브에이전트가 대화를 분석하여 4가지 타입으로 메모리를 자동 추출하고, 다음 세션에서 관련 메모리만 선별 로드합니다.

40개 이상의 도구가 권한 검증과 동시성 제어를 거쳐 스트리밍 실행되는 과정

Layer 2: AI 분류기LLM 판단 + Bash AST 파싱 (tree-sitter)
Layer 3: 사용자 확인ALLOW / DENY / ASK 다이얼로그

OUTPUT
