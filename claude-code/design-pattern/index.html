Prompt Assembly Pipeline

명령어 해석 Instruction Following

입력된 프롬프트가 15개 섹션으로 분해되어 시스템 프롬프트로 조립됩니다. 정적 영역(7개)은 전역 캐싱되고, 동적 영역(8개)은 세션별로 갱신됩니다.

Input prompts are decomposed into 15 sections and assembled into system prompts. Static sections (7) are globally cached, dynamic sections (8) are refreshed per session.

02
Multi-layer Knowledge Stack

컨텍스트 메모리 Context Memory

대화 히스토리와 지식이 다층 메모리 스택에 축적됩니다. CLAUDE.md 계층, AI 관리 메모리, 세션 메모리, 자동 추출이 유기적으로 연동됩니다.

Conversation history and knowledge accumulate in a multi-layer memory stack. CLAUDE.md hierarchy, AI-managed memory, session memory, and auto-extraction work together.

Multi-Agent Orchestration

계획 수립과 추론 Planning & Reasoning

복잡한 문제를 코디네이터가 3단계로 분해합니다: 탐색(병렬) → 종합 → 구현/검증. 6가지 특화 에이전트가 역할을 분담하여 협업합니다.

Complex problems are decomposed by the Coordinator in 3 phases: Research (parallel) → Synthesis → Implementation/Verification. 6 specialized agents collaborate with role-based division.

Streaming Tool Executor

도구 및 코드 실행 Tool & Code Execution

40개 이상의 도구가 스트리밍 실행기를 통해 동시성 제어와 권한 게이트를 거쳐 실행됩니다. Bash 명령은 tree-sitter AST 파싱으로 보안 검증됩니다.

Context Compaction Pipeline

정제와 출력 Refinement & Output

응답이 생성되고 컨텍스트 압축, 도구 결과 요약, 후처리 훅을 거쳐 최종 출력됩니다. 9항목 보존 전략으로 핵심 정보를 유지하며 컨텍스트를 압축합니다.
