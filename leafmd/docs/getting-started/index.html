설치Installation

시스템 요구사항System Requirements

  • macOS 12.0 (Monterey) 이상or later
  • Apple Silicon (M1+) 및 Intel 모두 지원Apple Silicon (M1+) and Intel supported
  • 디스크 공간: ~80MBDisk space: ~80MB
  • Windows/Linux 지원 예정Windows/Linux support planned

다운로드 페이지에서 .dmg 파일을 받아 Applications 폴더로 드래그하면 된다. 설치 후 첫 실행 시 macOS 보안 경고가 뜰 수 있는데, 시스템 설정 > 개인정보 보호 > 보안에서 "확인 없이 열기"를 눌러주면 된다.Download the .dmg file and drag it to your Applications folder. On first launch, macOS may show a security warning — go to System Settings > Privacy > Security and click "Open Anyway."

첫 문서 만들기Creating Your First Document

1

폴더 열기Open a Folder

Cmd+O로 작업 폴더를 연다. Leaf는 Obsidian처럼 폴더 단위로 문서를 관리한다. 어떤 폴더든 열 수 있다.Press Cmd+O to open a working folder. Like Obsidian, Leaf manages documents folder-by-folder. Any folder works.

2

새 문서 생성Create a New Document

Cmd+N으로 새 마크다운 파일을 만든다. 파일명을 입력하면 .md 확장자가 자동으로 붙는다.Press Cmd+N to create a new markdown file. The .md extension is added automatically.

3

바로 쓰기 시작Start Writing

마크다운을 입력하면 즉시 렌더링된다. # 제목을 치면 바로 큰 제목이 되고, **볼드**는 즉시 굵게 보인다. 미리보기 패널이 따로 없다 — 쓰는 곳이 곧 보는 곳이다.Type markdown and it renders instantly. # Heading becomes a heading immediately, **bold** appears bold right away. No preview pane — where you write is where you read.

4

자동 저장Auto-Save

1초 후 자동으로 저장된다. Cmd+S도 되지만, 거의 쓸 일이 없다.Auto-saved after 1 second of inactivity. Cmd+S works too, but you'll rarely need it.

문서 연결하기: 위키링크Connecting Documents: Wikilinks

[[문서이름]]을 입력하면 다른 문서로의 링크가 생긴다. 존재하지 않는 문서를 링크하면 클릭 시 자동으로 생성된다. 호버하면 미니 그래프와 문서 미리보기가 팝업된다.Type [[document-name]] to create a link to another document. If the document doesn't exist, clicking it creates it automatically. Hovering shows a mini graph and document preview.

그래프 뷰Graph View

Cmd+G로 지식 그래프를 연다. 위키링크로 연결된 문서들이 D3.js 기반 인터랙티브 그래프로 시각화된다. 노드를 드래그하고, 줌하고, 클릭하면 해당 문서로 이동한다.Press Cmd+G to open the knowledge graph. Documents connected via wikilinks are visualized as an interactive D3.js graph. Drag nodes, zoom, and click to navigate.

소스 모드 전환Source Mode Toggle

Cmd+/로 라이브 프리뷰와 소스 코드 모드를 전환할 수 있다. 라이브 프리뷰 모드에서는 마크다운이 즉시 렌더링되고, 소스 모드에서는 원본 마크다운 텍스트를 직접 본다.Toggle between live preview and source code mode with Cmd+/. Live preview renders markdown instantly, while source mode shows raw markdown text.

템플릿으로 빠르게 시작Quick Start with Templates

새 문서를 만들 때 40가지 이상의 내장 템플릿을 선택할 수 있다. 주간 플래너, 보고서, 학술 논문, 코넬 노트, 회의록 등 다양한 형식이 준비되어 있다. 모든 템플릿은 {{date}} 자동 치환을 지원한다.When creating a new document, choose from 40+ built-in templates. Weekly planners, reports, academic papers, Cornell Notes, meeting minutes and more. All templates support {{date}} auto-substitution.

내보내기Exporting

Cmd+E로 내보내기 메뉴를 연다:Press Cmd+E to open the export menu:

  • PDF — A3/A4/A5/Letter/Legal, 가로/세로 설정A3/A4/A5/Letter/Legal, portrait/landscape
  • HTML — 인라인 스타일 포함, 독립 실행 가능Inline styles, self-contained
  • DOCX — Microsoft Word 형식Microsoft Word format
  • HWP — 한글 워드프로세서 형식Korean Hangul word processor format

가져오기Importing

다른 형식의 파일을 마크다운으로 변환해서 가져올 수 있다:Convert other file formats into markdown:

  • .xlsx → 마크다운 테이블Markdown table
  • .pptx → 슬라이드별 마크다운Markdown per slide
  • .docx → 마크다운 변환Markdown conversion
  • .pdf → 텍스트 추출 후 마크다운Text extraction to markdown
  • .hwp → 마크다운 변환Markdown conversion

웹 페이지 → 마크다운 스크래핑Web Page → Markdown Scraping

내장 브라우저 패널에서 웹 서핑을 하다가 "Import" 버튼을 누르면, 현재 페이지의 DOM을 Turndown.js로 추출하여 깔끔한 마크다운 문서로 변환한다. 브라우저에서 텍스트를 복사(Cmd+C)하기만 해도 HTML이 자동으로 마크다운으로 변환되어 클립보드에 들어간다. Microsoft의 markitdown CLI를 통한 고급 문서 변환(Readability 추출)도 지원한다.In the built-in browser panel, click "Import" while browsing any web page — the DOM is extracted via Turndown.js and converted to clean markdown. Even copying text (Cmd+C) from the browser auto-converts HTML to markdown in clipboard. Advanced document conversion via Microsoft's markitdown CLI (Readability extraction) is also supported.
