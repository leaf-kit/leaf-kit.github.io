모드 & 패널Modes & Panels

상황에 맞는 작업 환경을 선택한다. 각 모드는 UI 요소를 다르게 구성하여 집중력을 높인다.Choose the right environment for the task. Each mode configures UI elements differently to enhance focus.

편집 모드Editing Modes

✍️

노멀 모드Normal Mode Cmd+Shift+E

기본 모드. 사이드바, 툴바, 상태바가 모두 보인다. 일반적인 문서 작업에 적합.Default mode. Sidebar, toolbar, and status bar all visible. Suitable for regular document work.

🎯

포커스 모드Focus Mode Cmd+Shift+F

사이드바와 툴바를 숨기고 에디터만 남긴다. 긴 글을 쓸 때 방해 요소를 제거한다.Hides sidebar and toolbar, leaving only the editor. Removes distractions for long writing sessions.

✋

헤밍웨이 모드Hemingway Mode Cmd+Shift+W

Delete 키를 비활성화한다. 지우지 않고 앞으로만 쓰게 강제한다. 분당 글쓰기 속도(WPM)와 타이머가 표시되어 글쓰기 생산성을 측정할 수 있다. "Write drunk, edit sober"의 정신.Disables the Delete key. Forces you to write forward without erasing. Shows words-per-minute (WPM) and a timer to measure writing productivity. The spirit of "Write drunk, edit sober."

📖

읽기 모드Reading Mode Cmd+Shift+G

읽기 전용 모드. 편집 불가능하고, 페이지 넘기기 방식으로 문서를 탐색한다. 싱글/듀얼 페이지 레이아웃을 지원.Read-only mode. No editing, page-flip navigation. Supports single and dual-page layouts.

🧘

미니멀 모드Minimal Mode Cmd+Shift+M

모든 UI 크롬을 숨기고 에디터만 남긴다. 포커스 모드보다 더 극단적. 글쓰기에만 완전히 몰입.Hides all UI chrome, leaving only the editor. More extreme than Focus mode. Complete immersion in writing.

🎬

프레젠테이션 모드Presentation Mode

마크다운 문서를 슬라이드로 변환한다. --- 수평선이 슬라이드 구분자가 된다. 전체 화면으로 발표 가능.Converts markdown to slides. --- horizontal rules become slide separators. Full-screen presentation ready.

전문가 모드Professional Modes

특정 작업 유형에 최적화된 UI 구성:UI configurations optimized for specific workflows:

  • 개발자 모드Developer Mode — 터미널 패널 + 개발 도구 통합Terminal panel + dev tools integration
  • 저널리스트 모드Journalist Mode — 5W1H 분석, 역피라미드 구조 검사, 수동태 탐지, 긴 문장 경고, 중복 단어 감지, 데드라인 타이머5W1H analysis, reverse pyramid check, passive voice detection, long sentence warning, duplicate word finder, deadline timer
  • 프롬프트 엔지니어 모드Prompt Engineer Mode — AI 프롬프트 최적화 도구AI prompt optimization
  • 아카데미 모드Academy Mode — Q&A 플래시카드 + 퀴즈Q&A flashcards + quiz system
  • 다이어리 모드Diary Mode — 날짜별 일기 + 기분 트래킹Date-stamped journal + mood tracking
  • 시니어 모드Senior Mode — 큰 글씨 + 단순화된 UI (접근성)Large fonts + simplified UI (accessibility)
