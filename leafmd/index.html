나는 복잡한 것을 좋아하지 않는다.I don't like complexity.

그래서 오히려 복잡한 문제를 해결하는 도구를 만든다.That's exactly why I build tools that solve complex problems.

글을 쓰기 위한 도구가 아니라,
생각이 자연스럽게 흐르고 실행까지 이어지게 만드는 환경을 만드는 중이다.Not a tool for writing,
but an environment where thoughts flow naturally and lead to action.

개인 지식 관리는 거창하지 않다.
잘 정리된 시스템보다,
다시 열어보고 싶어지는 기록이 더 중요하다고 믿는다.Personal knowledge management doesn't have to be grand.
A well-organized system matters less
than notes you actually want to come back to.

짧게 쓰고, 자주 남기고, 바로 실행한다.Write short. Save often. Act immediately.

Leaf는 아직 완성되지 않았다. 하지만 생각이 흘러가는 방식은 이미 바뀌고 있다.Leaf isn't finished yet. But the way thoughts flow has already changed.

기능Features

마크다운 에디터의 새로운 기준A New Standard for Markdown

필요한 것은 모두, 기대하지 못한 것까지Everything you need, and what you didn't expect

✍️

라이브 프리뷰Live Preview

Typora 스타일 인라인 WYSIWYG. 별도 미리보기 패널 없이 쓰는 그대로 보인다. Cmd+/ 로 소스 모드 전환.Typora-style inline WYSIWYG. No split pane — what you type is what you see. Toggle source with Cmd+/.

🕸️

지식 그래프Knowledge Graph

[[위키링크]]로 문서를 연결하면 D3.js 기반 인터랙티브 그래프가 자동 생성된다. 로컬/글로벌 뷰 전환.Connect documents with [[wikilinks]] and get an interactive D3.js graph. Local and global views.

📐

머메이드 다이어그램Mermaid Diagrams

플로차트, 시퀀스, 클래스, ER 다이어그램을 마크다운 안에서 바로 렌더링. Excalidraw 변환 지원.Flowcharts, sequences, class diagrams, ER diagrams rendered inline. Excalidraw conversion available.

📝

코넬 노트Cornell Notes

Leaf만의 코넬 노트 포맷. Cue | Note | Summary 3단 레이아웃을 마크다운 안에서 직접 편집.Leaf-exclusive format. Edit Cue | Note | Summary 3-column layout directly in markdown.

📋

칸반 보드Kanban Board

에디터를 떠나지 않고 인라인 칸반 보드로 작업을 관리한다. 메모 보드도 함께 지원.Manage tasks with inline Kanban boards. Memo boards also supported.

🌐

웹 → 마크다운 스크래핑Web → Markdown Scraping

내장 브라우저에서 웹 서핑 중 버튼 하나로 페이지를 마크다운으로 변환한다. Turndown.js 기반 DOM 추출. 복사만 해도 HTML이 마크다운으로 클립보드에 들어간다.Browse the web in the built-in browser and convert any page to markdown with one click. Turndown.js DOM extraction. Even copying auto-converts HTML to markdown in clipboard.

📊

KaTeX 수식KaTeX Math

인라인 $...$ 과 블록 $$...$$ 수식을 실시간 렌더링. 수학 기호 선택기 패널(123KB 사전) 내장.Real-time inline and block equation rendering. Built-in math symbol picker panel (123KB dictionary).

🎨

60가지 이상의 테마60+ Themes

GitHub, Forest, Aurora, Ocean, Retro PC, Win98, 손글씨, 펭귄, 아리랑 등. CSS 변수 기반 커스터마이징.GitHub, Forest, Aurora, Ocean, Retro PC, Win98, Handwriting, Penguin, Arirang and more. CSS variable customization.

📁

Git 패널Git Panel

에디터 안에서 Git 상태 확인, 스테이징, 커밋 히스토리 조회. git2-rs 네이티브 Rust 바인딩으로 빠르게.View Git status, staging, commit history inside the editor. Native Rust binding via git2-rs for speed.

📝

테이블 수식Table Formulas

마크다운 테이블에서 합계, 평균, 최대, 최소를 계산한다. 셀 클릭 편집, 우클릭 행/열 추가 삭제까지.Calculate sum, average, max, min in markdown tables. Click-to-edit cells, right-click to add/delete rows and columns.

🍅

뽀모도로 + ASMRPomodoro + ASMR

25/5분 뽀모도로 타이머 내장. 집중할 때 모닥불, 파도, 빗소리, 바람, 숲, 시냇물, 천둥, 화이트노이즈 8종 ASMR을 Web Audio API로 재생.Built-in 25/5min Pomodoro timer. 8 ASMR ambient sounds (campfire, ocean, rain, wind, forest, stream, thunder, white noise) via Web Audio API.

🏷️

태그 관리 시스템Tag Management

프론트매터 태그를 자동 수집, 파일을 태그별로 분류. 태그 통계, 미태그 파일 탐색, 자동 완성 제안까지.Auto-collect frontmatter tags, group files by tag. Tag stats, untagged file browser, auto-complete suggestions.

⌨️

커맨드 팔레트Command Palette

Cmd+K로 모든 명령을 검색하고 실행. VS Code처럼 키보드만으로 에디터의 모든 기능에 접근.Search and run any command with Cmd+K. Access every feature with keyboard only, like VS Code.

🌐

다국어 지원Multilingual

한국어, 영어, 중국어, 일본어 완전 지원. 한글 번호 매기기(가나다, 일이삼), 한자 사전 팝업까지.Full Korean, English, Chinese, Japanese. Hangul numbering, Hanja dictionary popup included.

저널리스트 모드Journalist Mode

5W1H 분석, 역피라미드 구조 검사, 긴 문장 감지, 수동태 탐지, 중복 단어 경고. 글쓰기 품질 분석기.5W1H analysis, reverse pyramid check, long sentence detection, passive voice finder, duplicate word warnings.

🎮

타이핑 게임 + 코드 배틀Typing Game + Code Battle

4가지 모드(단문, 장문, 키워드, 문장 재배열)로 타이핑 연습. Hot Coder 모드에서는 30개 알고리즘 코드를 AI와 WPM 대결.4 modes (short, long, keywords, reorder). Hot Coder mode: compete against AI typing 30 algorithm snippets.

🐣

다마고치Tamagotchi

배고픔, 행복, 에너지, 청결도 4가지 스탯. 8가지 감정과 애니메이션. 글을 쓰면 경험치가 쌓여 레벨업.4 stats: hunger, happiness, energy, cleanliness. 8 emotions + animations. Write to earn XP and level up.

🎬

시네마 스크린세이버Cinema Screensaver

52개 명언 + 5가지 모드(명언, 위스퍼, 시네마, 텍스트댄스, 워터폴). 눈, 비, 낙엽, 오로라 파티클 효과.52 quotes + 5 modes (quotes, whisper, cinema, textdance, waterfall). Snow, rain, leaves, aurora particles.

✋

헤밍웨이 모드Hemingway Mode

Delete 키를 비활성화. 지우지 말고 앞으로만 써라. WPM + 타이머 표시.Disables Delete key. Only move forward. Shows WPM + timer.

🌊

물리 시뮬레이터Physics Simulator

포물선 운동, 단진동, 파동 간섭, 원운동, 진자, 빗면, 충돌. KaTeX 수식 라벨까지. 교육용 시각화 7종.Projectile, SHM, wave interference, circular motion, pendulum, inclined plane, collision. 7 educational visualizations.

🌿

잔디 히트맵Grass Heatmap

GitHub 잔디처럼 파일 활동을 날짜별 히트맵으로 시각화. 생성/수정/접근 3가지 모드. 프론트매터 필터링.GitHub-style activity heatmap by date. Created/modified/accessed modes. Frontmatter filtering.

🎵

ABC 악보ABC Music Notation

ABC 문법으로 악보 SVG 렌더링 + MIDI 재생. BPM 조절, 노트 하이라이트, 25~400% 속도 워프.ABC notation to SVG + MIDI playback. BPM control, note highlighting, 25-400% speed warp.

🎓

플래시카드 학습Flashcard Quiz

마크다운에서 Q&A를 추출해서 플래시카드, 퀴즈 모드로 학습. 정답률 추적. 셔플 기능.Extract Q&A from markdown for flashcards and quizzes. Accuracy tracking. Shuffle support.

📔

다이어리 + 기분 트래킹Diary + Mood Tracker

8가지 기분 이모지, 연속 기록 스트릭, 감사일기 프롬프트. 365일 히스토리.8 mood emojis, consecutive streak tracking, gratitude prompts. 365-day history.

💭

글쓰기 타이머Writing Timer

자유 글쓰기, 아이디어 덤프, 요약, 논증 등 8가지 주제별 세션. 시간 내 글자 수를 추적하고 기록을 저장.8 topic sessions: freewriting, idea dump, summary, argument, etc. Tracks character count with session history.

🍂

낙엽 타이핑 게임Falling Leaves Game

화면 위에서 단어가 떨어진다. 맞춰 타이핑하면 점수. 8스테이지, 콤보 시스템, 스코어보드. 스테이지마다 속도 증가.Words fall from the top. Type them before they drop. 8 stages, combo system, scoreboard. Speed increases per stage.

🔊

텍스트 음성 변환Text-to-Speech

문서를 소리 내어 읽어준다. 속도 조절, 일시정지/재개. 교정이나 리뷰에 유용.Read your document aloud. Speed control, pause/resume. Useful for proofreading and review.

🧓

시니어 모드Senior Mode

큰 글씨(20px+), 단순화된 UI, 읽기 시간 추정. 접근성을 진지하게 고민한 결과물.Large fonts (20px+), simplified UI, reading time estimation. A serious take on accessibility.

💻

내장 터미널 + 개발 도구Terminal + Dev Tools

xterm 터미널, 색상 선택기, HTTP 클라이언트, JSON 포매터, 정규식 테스터, URL 디코더까지 DevToolkit 내장.xterm terminal, color picker, HTTP client, JSON formatter, regex tester, URL decoder — full DevToolkit built in.
