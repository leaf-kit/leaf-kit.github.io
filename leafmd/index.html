핵심 기능 Core Features

마크다운 에디터에 필요한 것, 그리고 기대하지 못한 것까지 What you need in a markdown editor, and what you didn't expect

✍️

라이브 프리뷰 Live Preview

Typora 스타일의 인라인 WYSIWYG. 별도 미리보기 패널 없이 쓰는 그대로 보인다. Cmd+/ 로 소스 모드 전환. Typora-style inline WYSIWYG. No separate preview pane — what you type is what you see. Toggle source mode with Cmd+/.

🕸️

지식 그래프 Knowledge Graph

[[위키링크]]로 문서를 연결하면 D3.js 기반 인터랙티브 그래프가 자동 생성된다. 로컬/글로벌 뷰 전환 가능. Connect documents with [[wikilinks]] and an interactive D3.js graph is auto-generated. Switch between local and global views.

📐

머메이드 다이어그램 Mermaid Diagrams

플로차트, 시퀀스, 클래스, ER 다이어그램을 마크다운 안에서 바로 렌더링. Excalidraw 변환도 가능. Render flowcharts, sequences, class diagrams, ER diagrams inline. Excalidraw conversion available.

📝

코넬 노트 Cornell Notes

Leaf만의 코넬 노트 포맷. Cue | Note | Summary 3단 레이아웃을 마크다운 안에서 직접 편집. Leaf-exclusive Cornell Notes format. Edit Cue | Note | Summary 3-column layout directly in markdown.

📋

칸반 보드 Kanban Board

에디터를 떠나지 않고 인라인 칸반 보드로 작업을 관리한다. 메모 보드도 함께 지원. Manage tasks with inline Kanban boards without leaving the editor. Memo boards also supported.

🎨

17가지 테마 17 Themes

GitHub, Night, Forest, Aurora, Ocean, Moonlight 등 라이트/다크 테마. CSS 변수 기반으로 커스터마이징 가능. GitHub, Night, Forest, Aurora, Ocean, Moonlight and more. Customizable with CSS variables.

📊

KaTeX 수식 KaTeX Math

인라인 $...$ 과 블록 $$...$$ 수식을 실시간 렌더링. 학술 논문과 기술 문서 작성에 적합. Real-time rendering of inline $...$ and block $$...$$ equations. Perfect for academic and technical writing.

📁

멀티 탭 Multi-Tab

여러 문서를 동시에 열어서 작업. 파일 트리 + 탭 기반 탐색. 외부 파일 변경도 실시간 감지. Work with multiple documents simultaneously. File tree + tab navigation. Real-time external file change detection.

🌐

다국어 지원 Multilingual

한국어, 영어, 중국어, 일본어 완전 지원. 한글 번호 매기기(가나다, 일이삼)까지 네이티브로 처리. Full Korean, English, Chinese, Japanese support. Native Hangul numbering (가나다, 일이삼) included.

예상 밖의 기능들 Unexpected features

🎮

타이핑 게임 Typing Game

키보드 속도를 연습하는 미니 게임이 내장되어 있다. 글쓰기 워밍업용. Built-in typing speed mini-game. Perfect for writing warm-ups.

🐣

다마고치 Tamagotchi

글을 쓰면 성장하는 가상 펫. 글쓰기 동반자 역할을 한다. A virtual pet that grows as you write. Your writing companion.

🎬

스크린세이버 모드 Screensaver Mode

52개의 영감을 주는 문구가 화면에 순환 표시된다. 사무실에 켜놓으면 분위기가 달라진다. 52 inspirational quotes rotate on screen. Leave it on in the office for ambient inspiration.

✋

헤밍웨이 모드 Hemingway Mode

Delete 키를 비활성화한다. 지우지 말고 앞으로만 써라. 분당 글쓰기 속도도 측정. Disables the Delete key. Don't erase — only move forward. Tracks your words per minute.

🌊

물리 시뮬레이터 Physics Simulator

인터랙티브 물리 캔버스가 들어있다. 마크다운 에디터에 왜? 만들고 싶어서. Interactive physics canvas is built in. Why in a markdown editor? Because we wanted to.

🎵

ABC 악보 ABC Music Notation

ABC 문법으로 악보를 렌더링하고 재생까지 할 수 있다. 음악 수업 노트에 제격. Render sheet music with ABC notation and play it back. Perfect for music class notes.

👨‍🎤

시니어 모드 Senior Mode

큰 글씨, 단순화된 UI. 접근성을 진지하게 고민한 결과물. Larger fonts, simplified UI. A serious take on accessibility.

💻

내장 터미널 Built-in Terminal

xterm 기반 터미널이 에디터 안에 있다. 문서 쓰다가 바로 명령어 실행. xterm-based terminal right in the editor. Run commands while writing docs.
